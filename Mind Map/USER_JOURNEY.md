# 🗺️ THE SENSES — USER JOURNEY MAP

## 1. Persona: "The Ambitious Developer" (Alex)

**Goal:** Flex their cognitive skills to get hired by top tech firms without a degree.

### Phase 1: Discovery & Onboarding

1. **Trigger:** Sees a tweet showing a friend's "Elite Mind" rank card.
2. **Action:** Visits `thesenses.ai` Landing Page.
3. **Thought:** *"This looks legit. Is it better than LeetCode?"*
4. **Interaction:** Clicks "Claim Your Rank".
5. **Sign Up:** Creates account with GitHub login.
6. **Profile Setup:** Selects "Full Stack Developer" as primary role.

### Phase 2: The Assessment Loop (High Stakes)

1. **Dashboard First View:** Empty rank, confidence 0.
2. **Action:** Clicks **"Start Assessment"**.
3. **Pre-Flight:**
    * Approves Webcam/Mic (Anti-Cheat).
    * Records short "Intro Pitch" video.
4. **The Test (15 Mins):**
    * **Logic:** Solves pattern rec problems.
    * **Focus:** Tracking cursor stability.
    * **Reflex:** Reaction time drills.
5. **Result Reveal:**
    * **Score:** 845 (Top 15%).
    * **Insight:** *"Your logic is strong, but attention wavered at min 12."*
6. **Emotion:** Pride + Urgency to improve.

### Phase 3: Social Proof & Growth

1. **Share:** Posts generated "Analyst Tier" card to Twitter.
2. **Growth:** Reviews "Cognitive Timeline" to see dips.
3. **Engagement:** Joins a **Bubble Discussion** on "AI Ethics".
    * Posts a thought.
    * Someone "Quotes" Alex into a new branch.
4. **Routine:** Returns daily to keep streak alive and improve Focus score.

## 2. Persona: "The Skeptical Recruiter" (Sarah)

**Goal:** Find candidates with high raw intelligence, ignoring generic CVs.

### Phase 1: Access & Setup

1. **Login:** Enters Company Dashboard.
2. **Context:** Needs a Junior Dev with high learning velocity.

### Phase 2: Talent Discovery

1. **Search:** Filters by `Role: Developer` + `Growth: >5%` + `Rank: Top 20%`.
2. **Result:** Sees 5 candidates. "Alex" stands out.
3. **Deep Dive (Alex's Profile):**
    * **Check:** Integrity Score is 98% (Low Cheat Risk).
    * **Check:** Consistency is High.
    * **Review:** Watches "Intro Pitch" video (15s).
    * **Ignore:** Doesn't see Alex's likes/followers (Distraction free).

### Phase 3: Decision

1. **Compare:** Adds Alex and another candidate to "Comparison Tool".
2. **Insight:** Alex has faster reflex and higher adaptability.
3. **Action:** Clicks "Request Interview".
